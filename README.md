# College-Course-Tracker

### The purpose of this project is to create an mobile application that can help students track their courses.
The application provides the ability to add new terms with corresponding courses, that have corresponding tests.
The objects are stored in an SQLite database using Room architecture. The Objects are shown in recycler 
views which have detailed information views. 

-----------------------------
### Features 
 - Secure log in page, with the ability to create new user
 - Add/Edit/Delete for Terms/Courses/Tests/Notes
 - Creates course notes, and share via messeges
 - Create alerts for upcoming Courses/Tests
 - Search for Courses/Tests by title
 - Custom views by Course/Test type
 - View all Terms/Courses/Tests 
 - Work in portrait or landscape mode
